---
title: Smoke test checklist
---

Use this list before every release.

Core flows
- Sign up and sign in
- Create a community and invite a member
- Join a community via invite
- Create an event and RSVP
- Create a task and mark it done
- Create a finance request and mark settled
- Post to the feed and comment

Admin flows
- Change member roles
- Update community settings
- Upload community profile image

Confirm there are no blocking issues before release.
