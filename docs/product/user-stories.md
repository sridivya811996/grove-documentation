These user stories are organized by functional area to keep scope clear and avoid duplication.

Authentication
- As a user, I want to sign in with Google so I can start quickly.
- As a user, I want to sign up with email so I can use Grove without Google.
- As a user, I want to stay signed in so I do not log in every time.

Community and membership
- As an admin, I want to create a community so I can organize my group.
- As a member, I want to join by invite so access stays controlled.
- As an admin, I want to approve join requests for public communities.
- As an admin, I want to manage members and roles.

Profiles
- As a member, I want to update my name and photo so others recognize me.
- As an admin, I want a community profile image for identity.

Events and meetings
- As an organizer, I want to create and edit events.
- As a member, I want to RSVP and see event details.
- As an organizer, I want to send reminders.
- As an admin, I want to capture meeting minutes and agenda items.

Tasks and activities
- As an admin, I want to create tasks for events or meetings.
- As a member, I want to see my assigned tasks and mark them complete.
- As an admin, I want task status visibility across the community.

Feed and engagement
- As a member, I want to post updates to the community feed.
- As a member, I want to comment on posts.
- As an admin, I want important updates to be visible without chat noise.

Finance
- As an admin, I want to request contributions from members.
- As a member, I want to mark payments as settled.
- As an admin, I want a clear view of outstanding contributions.

Notifications
- As a member, I want a single place to see all alerts.
- As a member, I want reminders for my tasks and events.

These stories define the MVP behavior and keep the focus on practical coordination features.
