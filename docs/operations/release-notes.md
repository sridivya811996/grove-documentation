# Release Notes

## Version 1.0.0 (Initial Release)

**Release Date**: February 2026

### What's New

Welcome to Grove! This is our first public release. Grove helps you organize and manage your communities with powerful yet simple tools.

#### Core Features

**Communities**
- Create communities for your groups, clubs, or organizations
- Invite members via unique invite codes
- Three membership roles: Owner, Admin, Member
- Customize community settings and features

**Events**
- Create events with title, description, date, time, and location
- Add location links (Google Maps, etc.)
- Members can RSVP: Going, Maybe, Not Going
- View attendee list
- Automatic email notifications for new events

**Community Feed**
- Post updates, announcements, and discussions
- Comment on posts
- Mention other members with @username
- Pin important posts (admins)

**Polls**
- Create polls with multiple options
- Real-time vote counting
- Set poll expiration dates
- View who voted for what

**Tasks**
- Create tasks for events or meetings
- Assign tasks to members
- Track status: Open, In Progress, Blocked, Done
- Set due dates and reminders

**Finance Tracking**
- Record income and expenses
- Categorize transactions
- Attach receipt images
- View transaction history
- Contribution requests for splitting costs

**Meetings**
- Create meeting notes
- Add action items
- Assign follow-ups to members
- Track completion

**Member Management**
- View all community members
- Promote members to admin
- Remove members from community
- Leave communities

**Notifications**
- Push notifications for events, messages, tasks
- Email notifications for events and invites
- Customizable notification preferences

**Profile**
- Edit name and username
- Upload profile photo
- Set preferred currency
- Manage notification settings

### Authentication
- Sign in with Google
- Sign in with Apple
- Secure session management

### Technical
- Built with React Native and Expo
- Supabase backend for real-time data
- Works offline with automatic sync
