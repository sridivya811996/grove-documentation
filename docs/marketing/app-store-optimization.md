---
title: App store optimization
---

Objectives
- Increase discoverability for community organizers
- Improve conversion from store page to install

Key actions
- Use benefit-driven titles and subtitles
- Prioritize screenshots that show events, tasks, and finance
- Encourage early users to leave reviews
- Keep descriptions simple and aligned with real features

Localization
- Start with English and add local languages when metrics justify it

Update cadence
- Refresh screenshots for major releases
- Keep store copy aligned with the roadmap
