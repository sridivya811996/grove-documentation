The Grove app uses a mobile-first stack with a managed backend to keep delivery fast, reliable, and cost-conscious.

Guiding principles
- Ship quickly without sacrificing reliability
- Keep the data model simple and extensible
- Support feature flags for optional modules
- Treat privacy and trust as first-class requirements

Core components
- Mobile client built with Expo and React Native
- Backend services powered by Supabase
- Media storage with controlled access
- Notification delivery for reminders and alerts

This section explains the architecture, data model, and key technology choices.
