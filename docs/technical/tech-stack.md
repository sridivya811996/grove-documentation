---
title: Technology stack and rationale
---

Mobile
- Expo + React Native
- Enables rapid iteration, consistent behavior, and shared code across iOS and Android

Backend
- Supabase (PostgreSQL, Auth, Storage)
- Reduces infrastructure overhead while keeping SQL flexibility and strong security controls

Notifications
- Push notifications for reminders and updates
- Email provider for invites and important alerts

Why this stack
- Fast MVP delivery without heavy DevOps
- Clear upgrade path as usage grows
- Large ecosystem and proven tooling

These decisions balance speed, reliability, and cost efficiency.
