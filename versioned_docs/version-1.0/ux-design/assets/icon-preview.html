Export Instructions

To generate production icons:

  • Open this file in a browser and take screenshots, OR
  • Use icon.kitchen for Android adaptive icons
  • Use appicon.co for all iOS/Android sizes
  • Use Figma to recreate this design and export at required sizes

Required files for Expo:

  • icon.png - 1024×1024 (main app icon)
  • adaptive-icon.png - 1024×1024 (Android foreground, icon centered in 432×432 safe zone)
  • splash-icon.png - 200×200 (splash screen icon)
  • notification-icon.png - 96×96 (Android, white silhouette on transparent)
  • favicon.png - 48×48 (web)
